# Base Palette Press

## App Name

Base Palette Press

## Tagline

Publish color like a poster

## Description

Publish a 3-color poster palette on Base with a title, caption, and permanent visual color study.

## Category

Creative utility / palette archive / visual publishing

## Short Pitch

Base Palette Press turns a simple three-color study into a bold poster card on Base, giving designers and taste-makers a lightweight onchain archive of visual direction.

## Production Environment Variables

- `NEXT_PUBLIC_PALETTE_PRESS_CONTRACT_ADDRESS`: deployed palette press contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
