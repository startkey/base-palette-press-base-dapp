# Base Palette Press

A pocket-sized Base app for color publishing. Not a platform, not a feed, not a giant dashboard. Open it, connect, pressing a palette, and walk away with a palette card.

## Start

```bash
npm install
npm run dev
```

Stack: Next.js, React, TypeScript, Tailwind CSS, wagmi, viem, Base, Vercel.

Do not commit `.env`, private keys, seed phrases, RPC keys, GitHub tokens, or Vercel tokens. Use `.env.example` only for placeholders.
